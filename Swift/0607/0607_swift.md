# 0607 Swift

- 이름 짓기 규칙 : Camel Case
  - Lower Camel Case : function, method, variable, constant
  - Upper Camel Case : type(class, struct, enum, extension...)
  - 대소문자 구분
- console.log
  - print : 단순 문자열 출력
  - dump : 인스턴스의 자세한 설명까지 출력
- 문자열 보간법
  - String Interpolation
  - 프로그램 실행 중 문자열 내에 변수 or 상수의 실질적인 값을 표현하기 위해 사용
  - \ (변수명)
